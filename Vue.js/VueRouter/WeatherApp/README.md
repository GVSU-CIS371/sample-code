# WeatherApp: Intro Vue Router Lesson

This repository is a Vue 3 + TypeScript + Vue Router demo.

The demo focuses on four concepts:

1. Route setup in `src/main.ts`
2. Rendering with `router-view` in `src/App.vue`
3. Navigation with `router-link` and `router.push`
4. Dynamic route params with `/region/:zipCode/:numDays`

## Run locally

```bash
npm install
npm run dev
```
